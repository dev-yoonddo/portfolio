🧩 문제 정의

긴 문서를 읽는 데 시간이 많이 소요되고, 핵심 내용을 빠르게 파악하기 어렵다는 문제를 해결하고자 서비스를 개발했습니다.

⚙️ 주요 기능

  • 텍스트 입력 후 자동 요약
  • 핵심 키워드 추출
  • 요약 결과 출력

🧠 기술적 구현

  • 요청 → 처리 → 응답 구조의 API 설계
  • 텍스트 처리 로직을 별도 서비스 레이어로 분리
  • 대량 텍스트 처리 시 성능 고려

🚧 문제 해결 경험

  • 긴 텍스트 입력 시 처리 속도 지연 문제 → 처리 로직 개선
  • 요약 결과 품질 개선을 위한 알고리즘 보완
